# PHPUNIT-TEST
PHPUnit is a programmer-oriented testing framework for PHP. It is an instance of the xUnit architecture for unit testing frameworks.
Sample:
Download phpunit from https://phpunit.de
Calculatortest.php
Create empty index.php 
To test calculatortest.php
Cd into directory in vscode
Run like this from command line: 
./vendor/bin/phpunit calculatortest.php from vscode
